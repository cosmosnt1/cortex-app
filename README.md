# 🎥 Cortex: Producer OS

El sistema nervioso central para la producción ejecutiva cinematográfica.
Cortex es una suite modular diseñada para optimizar la gestión administrativa y financiera de proyectos audiovisuales. Este repositorio contiene el MVP: **Cortex Liquidación**, una herramienta inteligente para la automatización de informes económicos según los estándares de DAFO (Ministerio de Cultura, Perú).

---

## 1. ✨ Características Principales

- `Extracción Inteligente (IA)`: Procesamiento de Facturas, Boletas y RHE mediante Gemini 1.5 Pro Vision.
- `Ecosistema Workspace`: Integración nativa con Google Drive para almacenamiento jerárquico y nomenclatura automática.
- `Native Grid`: Interfaz de liquidación de alto rendimiento con estética iOS Glassmorphism.
- `Multi-Tema`: Soporte nativo para Modo Light (rodajes de día) y Modo Dark (post-producción).

---

## 2. 🛠️ Stack Tecnológico

- `Frontend`: React + Tailwind CSS v4 + Framer Motion.
- `Backend`: Node.js + Firebase (Auth & Firestore).
- `IA`: Google Gemini API.
- `Storage`: Google Drive API.

---
